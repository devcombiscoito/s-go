# Código de Conduta

Este projeto adota o [Contributor Covenant](https://www.contributor-covenant.org/) versão 2.1.

Nota: a versão em inglês está no final.

## Nosso Compromisso
Prometemos tornar a participação em nossa comunidade uma experiência livre de assédio para todos.

## Nossos Padrões
Exemplos de comportamento que contribuem para um ambiente positivo:
- Usar linguagem acolhedora e inclusiva
- Respeitar pontos de vista diferentes
- Aceitar críticas construtivas

Exemplos de comportamento inaceitável:
- Assédio ou linguagem ofensiva
- Ataques pessoais ou comentários depreciativos
- Publicação de informações privadas

## Responsabilidade e Aplicação
Os mantenedores do projeto são responsáveis por esclarecer padrões e agir conforme necessário.

Se você testemunhar ou for alvo de comportamento inaceitável, entre em contato com o mantenedor do projeto.

## Atribuição
Este Código de Conduta é adaptado do Contributor Covenant, versão 2.1:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

# Code of Conduct (English)

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/) version 2.1.

## Our Pledge
We pledge to make participation in our community a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Using welcoming and inclusive language
- Respecting differing viewpoints
- Accepting constructive criticism

Examples of unacceptable behavior:
- Harassment or offensive language
- Personal attacks or derogatory comments
- Publishing private information

## Enforcement
Project maintainers are responsible for clarifying standards and taking action as needed.

If you witness or are subject to unacceptable behavior, please contact the project maintainer.
