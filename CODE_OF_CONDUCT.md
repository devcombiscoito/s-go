# Código de Conduta

Este projeto adota o [Contributor Covenant](https://www.contributor-covenant.org/) versão 2.1.

## Nosso Compromisso
Prometemos tornar a participação em nossa comunidade uma experiência livre de assédio para todos.

## Nossos Padrões
Exemplos de comportamento que contribuem para um ambiente positivo:
- Usar linguagem acolhedora e inclusiva
- Respeitar pontos de vista diferentes
- Aceitar críticas construtivas

Exemplos de comportamento inaceitável:
- Assédio ou linguagem ofensiva
- Ataques pessoais ou comentários depreciativos
- Publicação de informações privadas

## Responsabilidade e Aplicação
Os mantenedores do projeto são responsáveis por esclarecer padrões e agir conforme necessário.

Se você testemunhar ou for alvo de comportamento inaceitável, entre em contato com o mantenedor do projeto.

Este Código de Conduta é adaptado do Contributor Covenant, versão 2.1:
